This site is for St Louis based developers to maintain a profile based on their contributions to open source. The main objective is to allow for developers who are interested in certain technologies to find eachother. Also potentially companies can use this site to find developers. Rather than relying on recruiters who send blanket emails, companies and developers can interact in a more direct manor.

This site harvests public information from Github. Eventually there will be an option to opt out, but that isn't implemented yet.

If you would like to help development, fork on Github!
